# 🐋 ABYSS

> On-Chain Intelligence Terminal — surface alpha from the depths

Single-page landing site for ABYSS, an on-chain intelligence platform untuk crypto traders. Whale tracking, smart-money detection, anomaly alerts across 12+ chains.

## ✨ Features (landing page)

- Animated underwater theme (deep glow, particles, whale silhouettes, pulse rings)
- ASCII art hero with gradient typography
- Live terminal demo dengan whale alert, smart money signal, anomaly detection
- 6 feature cards (whale tracking, smart money, anomalies, flow mapping, webhooks, pattern library)
- 12 chain tags
- 4-step "how it works" flow
- 3-tier pricing (Surface free / Deepwater $49 / Abyssal $249)
- Stats bar (12 chains, 47K whales, 2.3s latency, $8.4B volume)
- Smooth scroll, no external assets except Google Fonts

## 🚀 Deploy

### Vercel drag-drop
1. Buka [vercel.com/new](https://vercel.com/new)
2. Drag folder `abyss/` ke drop area
3. Deploy. URL hidup dalam 30 detik.

### Local
```bash
cd abyss
python3 -m http.server 8000
# buka http://localhost:8000
```

## 🏗 Stack

- Single-file HTML, vanilla JS, zero dependencies (kecuali Google Fonts)
- Pure CSS animations (no library)
- Static deploy, kompatibel Vercel / Netlify / GitHub Pages / IPFS
